# Introduction

## What is Haskell?
Haskell is a purely functional, lazy, statically typed programming language.

- Purely Functional
    - Function are values
    - Values never change
- Lazy
- Statically Typed

## Why Should I Learn Haskell?

- Industrial Strength
- Performance
- Libraries
