In the Lists chapter, you wrote a Caesar cipher. Now, we want to
expand on that idea by writing a VigenÃ¨re cipher. A VigenÃ¨re cipher
is another substitution cipher, based on a Caesar cipher, but it
uses a series of Caesar ciphers for polyalphabetic substitution. The
substitution for each letter in the plaintext is determined by a fixed
keyword.
So, for example, if you want to encode the message â€œmeet at
dawn,â€ the first step is to pick a keyword that will determine which
Caesar cipher to use. Weâ€™ll use the keyword â€œALLYâ€ here. You repeat
the keyword for as many characters as there are in your original
message:
MEET AT DAWN
ALLY AL LYAL
Now the number of rightward shifts to make to encode each
character is set by the character of the keyword that lines up with it.
The â€™Aâ€™ means a shift of 0, so the initial M will remain M. But the â€™Lâ€™
for our second character sets a rightward shift of 11, so â€™Eâ€™ becomes
â€™Pâ€™. And so on, so â€œmeet at dawnâ€ encoded with the keyword â€œALLYâ€
becomes â€œMPPR AE OYWY.â€
